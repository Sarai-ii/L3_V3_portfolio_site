About Me

I love building at the intersection of design, code, marketing, and strategy—crafting experiences that are not only visually engaging but built to perform and make a real impact.

My journey began in business and marketing, where I learned to understand audiences, uncover insights, and think strategically. That curiosity naturally pulled me toward the creative and technical side—wanting to not just shape the what and why, but also build the how. While I still enjoy being a helping hand for business development, I wanted to create my own business. This led me to Pursuit, a full-stack software engineering bootcamp, where I’ve been turning ideas into real, functional products. This is where I found my passion for web development and design, and I haven’t looked back since. From the teamwork to the problem-solving, I thrive in collaborative environments where I can learn from others and share my own insights. I’m a firm believer in the power of empathy and communication, and I strive to bring those values into every project I work on.

Outside of tech, you’ll find me journaling, exploring new cafés and cities, bingeing on the latest fashion and beauty trends, or simply getting competitive during family board game nights. Nonetheless, I’m always learning—and always building toward something meaningful, even in my free time.

My Projects

Project 1: Happiness Exchange Project

Built a full-stack platform that connects users through themed meaningful events and shared experiences. Users can register, log in, and participate in events, with all user-event relationships managed via a PostgreSQL database. The team and I implemented backend CRUD operations, secured authentication with Firebase, and RESTful route handling to ensure a seamless and scalable experience.

  • Key Features:
  • User Authentication: Secure login and registration using Firebase Auth.
  • Event Management: Users can create, join, and manage events.
  • Responsive Design: Ensures a seamless experience across devices.
  • RESTful API: Efficient data handling and interaction.
  • PostgreSQL Database: Robust data storage and retrieval.

Project 2: What's Dish

A recipe and cocktail discovery app that lets users search for meals by name or ingredient, explore international cuisines, and find drink pairings—all powered by external APIs and built with vanilla JavaScript.

  • Key Features:
  • Search by Keyword: Users can find recipes by typing ingredients, dish names, or keywords.
  • Random Cuisine Generator: Automatically fetches diverse international dishes on page load.
  • Perfect Pairings: A cocktails page offers drinks that complement the meals.
  • Responsive Design: Seamless experience across desktop and mobile devices.

Project 3: Student Dashboard Project

Developed a responsive student dashboard using React, focusing on clean UI design and effective data presentation. The application displays organized, student data, ensuring clarity and usability.

  • Key Features:
  • Implemented component-based architecture for modularity.
  • Managed application state to handle dynamic data.
  • Utilized React Router for seamless navigation between views.
  • Intergrated CSS for responsive and accessible design.
